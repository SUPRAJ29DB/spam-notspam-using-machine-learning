# spam-notspam-using-machine-learning
📧 Email Spam Detection using Machine Learning

A professional Machine Learning web application that detects whether an email/message is Spam or Not Spam using Naive Bayes Algorithm and TF-IDF Vectorization.

Built with:

Python
Scikit-learn
Streamlit
NLP Techniques

⚙️ Installation

Clone the repository:

git clone https://github.com/SUPRAJ29DB/spam-notspam-using-machine-learning.git

Move into project folder:

cd spam-notspam

Install dependencies:

pip install -r requirements.txt
▶️ Run Application

Start Streamlit app:

streamlit run app.py
